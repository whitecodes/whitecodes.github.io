安装 zgen

原本就在用zsh，但是oh my Zsh是手动配置的，安装比较麻烦，而且手动安装code::stat比较麻烦。看网上的对比，在zplug和zgen中选了zgen。其实后来才发现大部分shell管理都是支持从网络上下载插件的

以下的记录都是以zgen为准，如果使用了其他的shell管理工具，请参考工具的主页。

下面的内容大部分是参考了zgen的主页
